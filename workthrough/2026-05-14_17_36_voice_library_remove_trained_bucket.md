# 나의 목소리들 훈련 모델 분리 기준 정리

## 개요
`나의 목소리들`에서 별도 `훈련한 모델` 탭을 제거하고, 각 모델 탭이 자기 모델의 프리셋과 학습 결과를 함께 관리하도록 정리했다. Qwen과 S2-Pro도 다른 모델과 같은 기준으로 보이게 맞췄다.

## 주요 변경사항
- 수정한 것: `훈련한 모델` 탭과 숨은 `trained` 탭 콘텐츠 제거
- 개선한 것: Qwen 탭에 Qwen 프리셋과 Qwen 파인튜닝 모델을 함께 표시
- 개선한 것: S2-Pro 탭에 저장 목소리와 S2-Pro 학습 실행 결과를 함께 표시

## 결과
- 빌드 성공: `npm run build`
- 코드 체크 성공: `git diff --check`
- 화면 확인: `나의 목소리들`에서 모델별 탭만 남고 별도 훈련 탭은 사라짐

## 다음 단계
- 각 모델 탭 안에서 `프리셋`, `학습 결과`를 카드 배지나 섹션 헤더로 더 선명하게 구분
