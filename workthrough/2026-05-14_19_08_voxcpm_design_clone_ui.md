# VoxCPM 목소리 디자인/복제 UI 정리

## 작업 내용
- VoxCPM 목소리 디자인 화면에서 목소리 설명과 실제 대사를 분리했다.
- VoxCPM 목소리 복제 화면에서 일반 복제와 이어 말하기 복제의 입력 순서를 다시 정리했다.
- voice design용 괄호 문법은 사용자가 직접 쓰지 않아도 실행 스크립트에서 자동으로 합치도록 바꿨다.

## 변경 사항
- 목소리 디자인:
  - `목소리 디자인 설명`과 `샘플 대사`를 별도 입력으로 분리.
  - 영어 설명 권장 문구를 사용자 흐름에 맞게 정리.
- 목소리 복제:
  - 일반 복제는 `복제할 목소리`만 필수로 보이게 정리.
  - 이어 말하기 복제는 `이어 말할 기준 음성` → `기준 음성의 대사` → 선택형 `추가 음색 참조` 순서로 정리.
- 생성 버튼 비활성 조건도 각 작업별 필수 입력에 맞게 조정.

## 검증
