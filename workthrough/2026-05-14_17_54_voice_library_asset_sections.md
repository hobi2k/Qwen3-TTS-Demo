# 나의 목소리들 자산 유형 섹션 분리

## 개요
모델별 탭 안에서 프리셋과 훈련 결과가 한 목록에 섞여 보이던 문제를 정리했다. 각 모델 탭 내부에 `훈련한 모델` 또는 `프리셋` 섹션 헤더와 카운트를 추가해 자산 종류를 바로 구분할 수 있게 했다.

## 주요 변경사항
- 개선한 것: Qwen 탭에 `훈련한 모델` 섹션과 `프리셋` 섹션 추가
- 개선한 것: S2-Pro 탭에 `훈련한 모델` 섹션과 `프리셋` 섹션 추가
- 개선한 것: VibeVoice LoRA/병합 모델을 `훈련한 모델` 계열로 명시
- 개선한 것: CosyVoice, VoxCPM, Supertonic, OmniVoice 탭에 `프리셋` 섹션 헤더 추가

## 결과
- 빌드 성공: `npm run build`
- 코드 체크 성공: `git diff --check`
- 화면 확인: Qwen 탭에서 훈련 모델과 프리셋이 별도 섹션으로 표시됨

## 다음 단계
- 각 모델의 실제 학습 결과 API가 추가되면 해당 모델 탭의 `훈련한 모델` 섹션에 연결
