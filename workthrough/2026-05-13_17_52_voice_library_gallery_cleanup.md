# 작업 기록: 목소리 라이브러리와 생성 갤러리 정리

## 처리한 내용
- `나의 목소리들`에서 `외부 모델` 탭을 제거하고 `프리셋/학습 결과` 탭으로 정리했습니다.
- VibeVoice 원본 베이스/ASR/런타임 모델은 목소리 자산 목록에서 제외하고, LoRA 학습 결과와 병합 모델만 표시하도록 바꿨습니다.
- CosyVoice, VoxCPM, Supertonic, OmniVoice는 저장된 프리셋만 `나의 목소리들`에서 재사용할 수 있게 유지했습니다.
- 생성 갤러리에 `모델별 생성` 필터를 추가해 VibeVoice, CosyVoice, VoxCPM, Supertonic, OmniVoice 생성 결과를 따로 확인할 수 있게 했습니다.
- 생성 갤러리 항목과 오디오 선택 입력 UI를 더 명확한 카드형 레이아웃으로 다듬었습니다.

## 검증
- `./node_modules/.bin/tsc --noEmit`
- `npm run build`

## 다음 개선 후보
- 각 모델별 프리셋 카드에 생성 샘플 미리듣기와 데이터셋 연결 상태를 더 직접적으로 노출하면 관리 흐름이 더 좋아질 수 있습니다.
