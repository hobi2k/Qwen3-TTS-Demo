# 표현 태그 선택 UI 추가

## 개요
Supertonic과 VoxCPM 생성 화면에 검색 가능한 태그 선택 UI를 추가했습니다. S2-Pro처럼 접이식 태그 영역에서 검색하고 누르면 현재 입력값에 바로 적용됩니다.

## 주요 변경사항
- 개발한 것: 공통 표현 태그 선택 컴포넌트 추가
- 개발한 것: Supertonic 공식 표현 태그를 대사에 바로 삽입
- 개발한 것: VoxCPM 영어 제어 지시문을 목소리 설명 또는 대사 앞 괄호 지시문으로 자동 삽입
- 개선한 것: README와 모델 가이드에 태그 적용 방식 설명 추가
