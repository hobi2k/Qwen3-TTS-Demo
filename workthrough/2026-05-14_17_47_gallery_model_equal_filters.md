# 생성 갤러리 모델별 필터 정리

## 개요
생성 갤러리에서 Qwen/S2-Pro만 별도 프리셋 필터로 보이고 나머지 모델이 `모델별 생성`으로 묶이던 구조를 수정했다. 생성 결과 필터도 `나의 목소리들`과 같은 기준으로 모델별 동등 탭처럼 보이게 정리했다.

## 주요 변경사항
- 수정한 것: `Qwen 프리셋 음성`, `S2-Pro 프리셋 음성`, `모델별 생성` 분류 제거
- 개선한 것: Qwen, S2-Pro, VibeVoice, CosyVoice, VoxCPM, Supertonic, OmniVoice를 같은 레벨의 생성 필터로 분리
- 개선한 것: 필터 버튼을 그리드로 바꿔 모든 분류가 화면에서 빠지지 않게 표시
- 수정한 것: 갤러리 설명 문구를 특정 모델 나열이 아니라 모델별/오디오 작업 기준 설명으로 변경

## 결과
- 빌드 성공: `npm run build`
- 코드 체크 성공: `git diff --check`
- 화면 확인: 생성 갤러리 상단에서 모델별 필터가 같은 레벨로 표시됨
