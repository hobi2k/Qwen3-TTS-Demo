# CosyVoice 참조 음성 선택 중복 제거

## 작업 내용
- CosyVoice 목소리 복제 화면에서 생성 갤러리 선택 UI가 중복으로 보이는 문제를 정리했다.
- 공통 오디오 입력 컴포넌트에 입력 방식 제한 옵션을 추가해, 필요한 입력에만 `경로`, `생성 갤러리`, `파일 선택` 탭을 노출할 수 있게 했다.

## 변경 사항
- `AudioSourceField`에 `modes` 옵션을 추가했다.
- CosyVoice voice conversion의 `변환할 원본 음성`은 경로 입력만 노출한다.
- CosyVoice `참조 음성`에는 기존처럼 생성 갤러리 선택을 유지한다.

## 검증
- `npm run build`
- `git diff --check`

## 다음 개선 후보
- 다른 모델의 원본 음성/참조 음성 입력도 같은 기준으로 점검해 중복 선택 UI가 생기지 않게 정리하면 좋다.
