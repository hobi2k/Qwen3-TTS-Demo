# 개발자용 문구 제거

## 개요
VoxCPM, CosyVoice, VibeVoice 화면과 문서에 남아 있던 내부 구현 중심 문구를 사용자용 설명으로 정리했습니다. 학습 흐름은 사용자가 이해할 수 있는 "데이터셋 선택, 학습 실행, 나의 목소리들에서 재사용" 기준으로 표현을 맞췄습니다.

## 주요 변경사항
- 수정한 것: VoxCPM 학습 안내와 경로 placeholder를 제품용 문구로 변경
- 수정한 것: README와 cookbook의 모델별 설명에서 내부 스크립트명과 세부 처리 단계명을 제거
- 개선한 것: VibeVoice 모델 도구의 학습 결과 선택 문구를 사용자가 이해하기 쉬운 표현으로 변경
