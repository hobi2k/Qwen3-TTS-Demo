# 작업 기록: 추가 모델 탭 UX 분리

## 처리한 내용
- CosyVoice에 `텍스트 음성 변환`과 `목소리 복제` 탭을 분리했습니다.
- VoxCPM에 `텍스트 음성 변환`, `목소리 디자인`, `목소리 복제` 탭을 분리했습니다.
- OmniVoice에 `텍스트 음성 변환`, `목소리 디자인`, `목소리 복제` 탭을 분리했습니다.
- 새 탭으로 진입하면 내부 task가 자동으로 해당 기능에 맞춰지도록 동기화했습니다.
- 각 모델 생성 화면 상단에 작업 카드형 전환 UI를 추가해 현재 작업 목적을 더 선명하게 보이도록 정리했습니다.
- 목소리 복제 화면에서만 참조 음성 선택 UI가 나오도록 OmniVoice 입력 레이아웃을 정리했습니다.

## 검증
- `./node_modules/.bin/tsc --noEmit`
- `npm run build`
- `git diff --check`

## 다음 개선 후보
- Supertonic도 `텍스트 음성 변환`과 `클로닝/스타일 생성`을 현재보다 더 시각적으로 분리하면 전체 모델군의 일관성이 더 좋아집니다.
